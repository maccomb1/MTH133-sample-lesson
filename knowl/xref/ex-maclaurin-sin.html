Example 1.1.6. (Together as Class) -- Maclaurin Series for \(\sin(x)\).

Find the first 5 terms of the Maclaurin series for \(f(x) = \sin(x)\text{.}\)
ðŸ”—
Answer.
\begin{equation*} \sin(x) = x - \frac{x^3}{6} + \frac{x^5}{120} + \cdots \end{equation*}
ðŸ”—
ðŸ”—
Solution.
This solution is for instructors only and would be removed for the student facing document. The intent is for this problem to be done in class together.
ðŸ”—
Let \(f(x)=\sin(x)\) and suppose a Maclaurin polynomial
ðŸ”—
\begin{equation*} P(x)=a_0 + a_1 x + a_2 x^2 + a_3 x^3 + a_4 x^4 + a_5 x^5 + \cdots \end{equation*}
We want the value and all matching derivatives to agree at \(x=0\text{.}\)
ðŸ”—
Match the value: We want \(f(0)=P(0)\) so
ðŸ”—
\(\sin(0)=a_0 + 0 + 0 + \cdots\) hence \(0=a_0\text{.}\)
ðŸ”—
Match the first derivative: We want \(f'(0)=P'(0)\text{.}\) Compute
ðŸ”—
\(f'(x)=\cos(x)\) and \(P'(x)=a_1 + 2a_2 x + 3a_3 x^2 + 4a_4 x^3 + 5a_5 x^4 + \cdots\text{.}\)
ðŸ”—
Plugging in \(x=0\) gives \(\cos(0)=0 + a_1 + 0 + 0 + \cdots\text{,}\) so \(1=a_1\text{.}\)
ðŸ”—
Match the second derivative: \(f''(x)=-\sin(x)\) and \(P''(x)=2a_2 + 6a_3 x + 12a_4 x^2 + 20a_5 x^3 + \cdots\text{.}\)
ðŸ”—
At \(x=0\text{:}\) \(-\sin(0)=2a_2\text{,}\) so \(0=2a_2\) and thus \(a_2=0\text{.}\)
ðŸ”—
Match the third derivative: \(f^{(3)}(x)=-\cos(x)\) and \(P^{(3)}(x)=6a_3 + 24a_4 x + 60a_5 x^2 + \cdots\text{.}\)
ðŸ”—
At \(x=0\text{:}\) \(-\cos(0)=6a_3\text{,}\) so \(-1=6a_3\) and \(a_3=-\dfrac{1}{6}\text{.}\)
ðŸ”—
Match the fourth derivative: \(f^{(4)}(x)=\sin(x)\) and \(P^{(4)}(x)=24a_4 + 120a_5 x + \cdots\text{.}\)
ðŸ”—
At \(x=0\text{:}\) \(\sin(0)=24a_4\text{,}\) so \(0=24a_4\) and \(a_4=0\text{.}\)
ðŸ”—
Match the fifth derivative: \(f^{(5)}(x)=\cos(x)\) and \(P^{(5)}(x)=120a_5 + \cdots\text{.}\)
ðŸ”—
At \(x=0\text{:}\) \(\cos(0)=120a_5\text{,}\) so \(1=120a_5\) and \(a_5=\dfrac{1}{120}\text{.}\)
ðŸ”—
Substituting these into \(P(x)\) gives, term by term,
ðŸ”—
\begin{equation*} \sin(x) =x - \frac{x^3}{6} + \frac{x^5}{120} - \cdots \end{equation*}
